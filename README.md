# Dev setup:

## Prerequisites:
- Install python 3
- Install nodejs

1. Clone repo
2. `cd` into directory
3. Create python venv with `python3 -m venv archive-bot`
4. Activate python venv with `source archive-bot/bin/activate`
3. Run `npm install`
4. Create `.env` file and get credentials from Karsten
5. Run `npm run dev`

Happy hacking...
